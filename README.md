# 🚀 TP Node.js – Gestion des Utilisateurs

## 📌 Description

Ce projet consiste à développer une application web complète en **Node.js** permettant de gérer une liste d’utilisateurs.

L’application inclut :

* Un **serveur HTTP**
* Une **API RESTful**
* Une **interface utilisateur simple en HTML/CSS**
* La **gestion des données utilisateurs**

---

## 🧰 Technologies utilisées

* **Node.js**
* **Express.js**
* **JavaScript (ES6)**
* **HTML / CSS**
* **Fetch API**
* **JSON**

---

## ⚙️ Fonctionnalités

### 👤 Gestion des utilisateurs

* ➕ Ajouter un utilisateur (nom, prénom, âge)
* 📄 Afficher la liste des utilisateurs
* 🔄 Mise à jour automatique après ajout

### 🌐 API REST

| Méthode | Endpoint     | Description                     |
| ------- | ------------ | ------------------------------- |
| GET     | `/api/users` | Récupérer tous les utilisateurs |
| POST    | `/api/users` | Ajouter un utilisateur          |
